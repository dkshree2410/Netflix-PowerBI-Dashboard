# 🎬 Netflix Power BI Dashboard

## 📌 Project Overview

This project presents an interactive **Netflix Analytics Dashboard** developed using **Microsoft Power BI**. The dashboard analyzes Netflix's content library and provides valuable insights into movies and TV shows through interactive visualizations and business intelligence techniques.

It demonstrates my skills in **Power BI, Power Query, DAX, data modeling, and data visualization** by transforming raw data into meaningful insights.

## 🚀 Features

- Interactive KPI Cards
- Rating-wise Content Analysis
- Genre-wise Distribution
- Country-wise Content Analysis
- Movies vs TV Shows Comparison
- Release Year Trends
- Director Analysis
- Interactive Filtering

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CSV Dataset

---

## 📈 Key Insights

- Analyzed over **3,200** Netflix titles.
- TV-MA is the most common content rating.
- Movies make up the majority of Netflix's library.
- Most content was released after 2015.
- The United States has the highest number of titles.
